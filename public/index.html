Using the DNA Features Extraction Tool
  1. Upload Sequence: Click on "Choose File" to select and upload a DNA FASTA file from your computer
  2. Select Features: Choose the features you would like to extract from the uploaded sequence(s)
  3. Extract Features: After selecting features, click on "Upload and Extract" to initiate the analysis process
  4. View Results: Once the analysis is complete, the extracted features will be displayed below.
